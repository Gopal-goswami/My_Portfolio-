# 🚀 Cyberpunk-Themed Professional Portfolio

A sleek, futuristic, and highly optimized personal portfolio website featuring a dynamic preloader welcome screen and tech-stack glow animations. Designed to showcase AI/ML capabilities, full-stack software engineering projects, and technical skills with an elite developer experience.

## 🌟 Key Features

*   **Futuristic Preloader Screen:** A custom cyber-black welcome overlay that manages asset loading times while delivering a premium first impression.
*   **Dynamic Progress Tracking:** Includes an animated loading bar and pulsing cyber-glow effects synced using performance-optimized CSS keyframes.
*   **Fully Responsive Architecture:** Clean and modular layout optimized across multiple viewports (Mobile, Tablet, and Desktop screens).
*   **SEO & Production Ready:** Structured semantic HTML markup ready for instant global CDN deployment.

## 🛠️ Tech Stack & Architecture

*   **Frontend:** Semantic HTML5, Advanced CSS3 (Flexbox/CSS Variables/Keyframes), Vanilla JavaScript (ES6+).
*   **Design Framework:** Custom Dark-Cyber theme with radial gradient fills and glow matrix filters.
*   **Typography & Vector Data:** Inter/Montserrat font sets integrated alongside FontAwesome technical icon packs.
